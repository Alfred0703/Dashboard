Connecting the Arduino Connect Arduino to the computer, and wait for connection status to change

Connect Arduino to the computer, and wait for connection status to change. In case the connection state does not change, you can verify that the correct COM port was detected in the Settings tab.

 Visualising the data Navigate to the Dashboard tab

Navigate to the Dashboard tab. There you can select which sensor you can view.

Due to the limitations of the Hardware, only one sensor can be viewed at a time.

Additional info can be seen in that tab, like the sensor id and time of the last measured data. This is only for development purposes only!

 Settings This tab is intended for chaging settings

Navigate to the Settings tab. You will be able to manually select a COM Port in case the automatic detection is not working.

You can change export setting of the data: CSV, JSON, or Excel.
